💡 How Service Pricing Works

Each item covers one service type. The agent answers pricing questions from live data — no hardcoding. Three response layers let you handle follow-up questions naturally. Items with Advisor Callback collect the caller's contact info for a specialist to call back.

Layer 1 — Primary answer ("How much is a service call?") Layer 2 — Follow-up ("Does that get credited?") Layer 3 — Deep follow-up ("What does it include?") Advisor Callback — Custom quotes via specialist call-back
